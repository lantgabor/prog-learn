## A programkód

- A számítógép ezt érti. Programozási nyelven írjuk.
- A nyelv elemei, **szótár**
- **Szintaxis**, nyelvtani szabályok.

```python
# legnagyobb közös osztó
a = int(input("a? "))
b = int(input("b? "))
 
while b != 0:
    t = b
    b = a % b
    a = t
 
print("lnko =", a)
```
