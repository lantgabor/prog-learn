1. 🧮 The Decimal System: What We're Used To

<img src="img/count.gif" className="float" width="35%" />

Humans love the number 10. With 10 fingers, our counting system—called decimal—is based on powers of 10:

```
Places:   10^2   10^1   10^0
Value:     100     10      1
Example:     3      2      5 → 3*100 + 2*10 + 5 = 325
```

It's how we think—but not how computers think.

Binary: The Language of Machines

<img src="svg/binary.svg" />

Computers use binary, a base-2 number system with only two digits: 0 and 1.
```
Places:   2^3  2^2  2^1  2^0
Value:      8    4    2    1
Binary:     1    0    1    1 → 1*8 + 0*4 + 1*2 + 1*1 = 11
```

Each 1 or 0 is a bit. 

8 bits = 1 byte. Think of a bit like a switch: ON or OFF.

Images:

<img src="img/Binary_counter.gif" width="40%" />

Összeadás:

Tízes számrendszerhez hasonlóan

```
   1011   (11 in decimal)
+  1101   (13 in decimal)
-------
 11000   (24 in decimal)
```

Pythonban is lehet:

```py
a = 0b1011
b = 0b1101
print(bin(a + b))  # 0b11000
```

Kivonás:

```
   1010   (10)
-  0011   ( 3)
-------
   0111   ( 7)
```

```py
a = 0b1010
b = 0b0011
print(bin(a - b))  # 0b111
```

Szorzás:

```
    101   (5)
  x  11   (3)
  ------
    101   (5 * 1)
+ 1010    (5 * 1 shifted)
-------
  1111   (15)
```

```py
a = 0b101
b = 0b11
print(bin(a * b))  # 0b1111
```

Osztás:
```
1100 ÷ 10  → 6
= 0b1100 ÷ 0b10 = 0b110
```
```py
a = 0b1100
b = 0b10
print(bin(a // b))  # 0b110
print(bin(a % b))   # 0b0 (remainder)
```
counting in binary on fingers is easy

Searching Names: Linear vs Binary

<img src="img/phonebook.jpg" width="20%" />

```
names = ["Adam", "Béla", "Cecil", "Dóri", "Erik"]
```

# Linear search

minden elso oldal

vagy minden masodik oldal?

```
for name in names:
    if name == "Dóri":
        print("Found!")
```

# Binary search

```python
import bisect
index = bisect.bisect_left(names, "Dóri")
if names[index] == "Dóri":
    print("Found (fast)!")
```

Linear = check one-by-one 🐢
Binary = divide and conquer ⚡ (needs sorted data)
